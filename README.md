
  # Echo Dialer MVP Design

  This is a code bundle for Echo Dialer MVP Design. The original project is available at https://www.figma.com/design/7KPWbkqHT21RN2sdfM58tV/Echo-Dialer-MVP-Design.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

## Pixel-perfect mode (overlay)

This repo includes a lightweight overlay tool to match the UI 1:1 to the reference screenshots.

**Hotkeys:**

- `Shift + O` — toggle overlay
- `Shift + [` / `Shift + ]` — switch overlay image
- `Shift + -` / `Shift + =` — adjust overlay opacity

You can also start the app with `?overlay=1` to enable the overlay immediately.
