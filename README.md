# Dial1 — Neo‑Brutal Sales Dialer

Single‑screen (no‑scroll) React + TypeScript sales dialer with neo‑brutalism styling and keyboard-first flow.

## Dev
- Install: `npm i`
- Run: `npm run dev`
- Typecheck: `npm run typecheck`
- Build: `npm run build`
- E2E: `npm run test:e2e`

## Keyboard shortcuts
- `C` call (tel:)
- `S` skip
- `D` demo booked (confetti)
- `N` focus notes
- `↑/↓` navigate queue
- `M` open Google Meet (`meet.google.com/new`)
